MA3-STYLE GRID MATRIX

革命性的分組與網格介面。參考 MA3 的邏輯，使用色彩視覺化區分燈具群組，並以座標化形式呈現佈局，支援多選與自由拖放，完全不需要手動計算位址。

智慧排位與分組： 透過色彩直觀表達分組關係，結合螺旋搜尋演算法自動避開衝突功能，讓編排完畢後可一鍵同步傳回 ETC 控台，大幅提升編程準確度。

SYNC & ARRANGEMENT

深度雙向 OSC 整合與智慧佈局。不只是發送指令，更能即時接收 ETC Eos 控台回傳的燈具選擇狀態，並支援從 Augment3d 自動同步位置，極速建立工作介面。

雙向聯動與快速排列： 系統可自動獲取控台 CH 資料與 3D 座標，同步生成網格位置，省去繁瑣的手動對位時間，達成「所見即所得」的編程體驗。

OPEN FADE

強大的色彩與漸變控制。快速將您製作的 GROUP 進行 FAN (扇形) 設定，大幅簡化複雜的燈具排列調整流程。

補足功能特效： 內建快速彩虹顏色功能，有效填補 ETC 控台原生的不足，讓您能一鍵渲染出流暢且精準的動態色彩效果。

EOS ECOSYSTEM ENGINE

專為 ETC Eos Family 打造，透過標準 UDP/OSC 協議達成無縫通聯。
支援 Layouts 存取功能，隨時切換不同場景的燈具配置。

TECHNICAL SPECS

  • OS: Windows 10 / 11 (Desktop Environment)
  • Communication: Bi-directional OSC over UDP
  • Ports: Out Port 8000 / In Port 8001 (Configurable)
  • Data Format: JSON Layout Management
  • Compatibility: ETC Eos v3.1+, Augment3d Support
