PRODUCT DEMO

VISUAL WAVEFORM

精準的視覺化波形分析。支援縮放功能，讓您即時看見音樂的起伏，點擊波形即可瞬間定位。

特色功能： 已標註的 Cue 點會直接顯示於波形圖上，讓您一目了然，精確掌握每一個節奏點。

PRECISION TIMING

支援 24, 25, 30 FPS 標準影格率。我們深知同步的重要性，因此提供了雙重時間顯示。

同步顯示： 同時呈現 FPS 影格時間 (Frame Time) 以及 音樂播放時間 (Actual Playback Time)，讓您在編程與回放時能有最直觀的對照。

ONE-CLICK MARK

極致簡化的工作流程。當您設定好燈控台 IP 與 Cue 號碼後，同步工作變得輕而易舉。

即時同步： 當音樂播放到預定位置時，只需按下 Mark，時間碼將即刻發送至 ETC 控台，瞬間完成複雜的時間碼寫入。

sACN DMX REMOTE

透過 sACN (E1.31) 協定接收 DMX 訊號，讓您可以直接從燈控台遙控播放器的 播放、暫停、停止、上一首與下一首等功能。

特色功能： 每個功能可自由設定對應的 DMX 通道 (1-512)，輕鬆實現燈光與音效的硬體連動控裝。

SMART PLAYLIST

自動化播放清單管理。專為多首曲目的演出設計，直觀的拖放與管理介面。

獨立設定： 每一首曲目皆可設定獨立的 Track Offset (起始時間碼)，切換曲目時自動套用，演出過程無縫接軌。

SEAMLESS INTEGRATION

內建 OSC 協議深度整合，一鍵 "Mark Cue" 直接寫入 ETC EOS 等燈控台。
同步支援 Hardware LTC Output，透過音訊介面輸出標準 SMPTE 訊號。

SYSTEM REQUIREMENTS

  • OS: Windows 10 / 11
  • Network: UDP Port 8000 (Firewall Allowed)
  • Audio: 48kHz Output Device (for LTC)
  • Format Support: MP3, WAV, OGG
  • Compatiblity: ETC EOS Family, Hog 4, Any OSC-enabled Console
